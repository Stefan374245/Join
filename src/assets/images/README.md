# Required Images for Auth Components

Please add the following image assets to this folder:

## Login & Signup Components:
- `mail.svg` - Email icon
- `lock.svg` - Password lock icon
- `visibility_off.svg` - Hide password icon (eye with slash)
- `person.svg` - Person/user icon (for signup name field)

## Optional:
- `joinlogodark.svg` - Join logo (dark version)
- `joinlogowhite.svg` - Join logo (white version)

## Icon Specifications:
- Format: SVG
- Size: 24x24px recommended
- Color: Should match the design (#2A3647 for primary icons)

You can find free icons at:
- Material Icons: https://fonts.google.com/icons
- Heroicons: https://heroicons.com/
- Bootstrap Icons: https://icons.getbootstrap.com/
